Pojects

Done

Potential profits for each of Companny's stores over the next six weeks

I developed a predictive sales algorithm for 1115 stores, which is available 24/7 via smartphone access. This model accurately predicts the potential profits for each of Companny's stores over the next six weeks and sends real-time information via API to the Telegram app. As part of this project, I also created a customized Telegram bot that accepts store codes, retrieves data from the API, and delivers sales forecasts instantly. This solution has enabled Companny to make data-driven decisions quickly and efficiently, resulting in improved sales and overall business outcomes.
